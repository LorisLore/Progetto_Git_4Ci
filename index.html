Il mio sito web

Il mio progetto

Questo sito web è stato realizzato come esercizio scolastico per imparare a usare GitHub e GitHub Pages.

Obiettivo

L'obiettivo del progetto è creare e pubblicare un sito web utilizzando un template HTML5UP.

Strumenti utilizzati

HTML, GitHub, GitHub Pages e un template HTML5UP.

Descrizione del sito

Il sito presenta una struttura grafica realizzata partendo da un modello base.

Attività svolta

L'attività svolta consiste nel modificare e pubblicare una pagina HTML online.

Pubblicazione

Il progetto è stato pubblicato tramite GitHub Pages.

Gestione del progetto

I file del sito sono stati gestiti tramite un repository GitHub.

Contenuto della pagina

La pagina contiene testi informativi relativi al progetto.

Apprendimento

Questo lavoro ha permesso di comprendere le basi della pubblicazione web.

Template grafico

Il layout grafico è stato ottenuto utilizzando il template HTML5UP.

Risultato ottenuto

Il risultato finale è un sito web funzionante e accessibile online.

Conclusione del lavoro

Il progetto rappresenta un primo approccio allo sviluppo di siti web.
